# Calculator (C#)

Bu proje C# dilinde geliştirilmiş basit bir console hesap makinesidir.  
Kullanıcıdan iki sayı alır ve dört temel matematik işlemini gerçekleştirir.

## Proje Amacı
Bu proje C# dilinde:
- Console input/output
- Değişken kullanımı
- Temel matematik işlemleri  
konularını pekiştirmek amacıyla geliştirilmiştir.

## Özellikler
- Toplama
- Çıkarma
- Çarpma
- Bölme

## Kullanılan Teknolojiler
- C#
- .NET Console Application

## Kurulum ve Çalıştırma
```bash
dotnet new console -n CalculatorApp -o . --force
dotnet run
